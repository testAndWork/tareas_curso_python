Bootstrap - Breadcrumbs

El breadcrumb (migas de pan) en Bootstrap es un componente de navegación que muestra la estructura jerárquica de una página web. Es útil para que los usuarios sepan dónde se encuentran en la jerarquía del sitio y puedan navegar fácilmente hacia atrás en la estructura. Bootstrap proporciona una clase llamada breadcrumb que puedes utilizar para crear este tipo de navegación.
