Uso de La Etiqueta < Input > y sus variantes

Tipo "Text"

  • Crea el campo de texto en una linea
  • Acepta cualquier tipo de texto alfanumerico

Tipo "Password"

  • Como el campo de textos, pero los caracteres se ocultan

Tipo "Email"

  • Acepta y Valida una direccion de correo

Tipo "Number"

  • Solo acepta numeros y puede incluir atributos como min, max step.

Tipo "Date"

  • Proporciona un selector de Fecha

Tipo "Checkbox"

  • Crea una casilla de verificacion que permite al usuario seleccionar una o varias opciones.

Tipo "Radio"

  • Crea una casilla de verificacion que permite al usuario seleccionar una opciones.

Tipo "File"

  • Permite Cargar un Archivo

Tipo "Submit"

  • Permite enviar un formulario.

Tipo "Reset"

  • Boton de reinicio que restaura los valores a los valores predeterminados.

Tipo "Hidden"

  • Crea un campo oculto que no es visible para el usuario, pero si
