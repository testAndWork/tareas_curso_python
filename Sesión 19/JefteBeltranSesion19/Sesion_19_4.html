Metodo POST y Get para envio de datos

Get

  • Se utiliza para solicitar datos del servidor
  • Los datos del formulario se agreguen a la URL como parametros de conusulta
  • Es visible en la barra de direcciones del navegador
  • Es adecuado para solicitudes que no cambien el estado del servidor(consulta a baase datos y busqueda
  • Los datos tienen una longitud limitada por las restricciones de longitud de la Url
    •

    Post

  • Envia datos del servidor para procesarlos
  • Los datos del formulario no son visibles
  • Es adecudado para enviar datos confidenciales y ademas puede ser mas extensos
  • No se utiliza para solicitudes que cambien el estado del servidor de ser asi se debe usar con precaucion
  • No tiene restricccion de longitud como el metodo Get
